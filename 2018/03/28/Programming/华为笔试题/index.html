华为笔试题，前面两道题非常简单就不进行记录了。

题目描述

输入是一个整型数组，数组里的整形数可以是正数，也可以是负数。子数组指的是数组中连续的一个或多个元素的集合，求这些子数组中，求和后最大值是多少。

输入描述：

一个整型数组

输出描述：

最大连续子数组的和。

编程

function GreatSum(arr){
    var sum = 0;
    var result = 0;
    for(var i in arr){
        sum += arr[i];
        if(sum < 0){
            sum = 0;
        }
        if(sum > result){
            result = sum;
        }
        if(result == 0){
            for(var j in arr){
                if(result == 0){
                    result = arr[j];
                } else if (result < arr[j]){
                    result = arr[j];
                }
            }
        }
    }
    return result;
}
